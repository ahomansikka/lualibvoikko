# lualibvoikko
Lualibvoikko is Lua (version 5.3) bindings for Libvoikko (voikko.puimula.org).

Folder C has the interface.

Folder lua5.3 has package Voikko.lua and a file of examples.
You can run the examples like this

lua5.3 examples.lua
